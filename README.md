> This project uses Node.js technologies that help manage files and create a dynamic display in the command panel to manage a personal to-do list.

## 💻 Prerequisites

Before you begin, please ensure you meet the following requirements:

- You have installed the latest version of `node.js`

- You have a `<Windows / Linux / Mac>` machine.

## 🚀 Installing readline-tasklist

To install `readline-tasklist`, follow these steps:

1. Download the .zip file directly from the GitHub repository [HERE](https://github.com/enzogran01/readline-tasklist).

2. Open bash/powershell in the readline-tasklist project path and run the following command:

```
node app.js
```
3. Done. Now you can manage your tasks!
